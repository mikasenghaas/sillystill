Many methods have presented as viable solutions for stylistic effect generation
using image to image translation using Deep Learning. Among these, there is a
variety of kinds of model architectures and effects produced for different
stylistic purposes.

\vspace{5pt}\noindent\textbf{Image Translation with GANs.} GANs have seen successful application in
the Image to Image domain, such as in Conditional Image-Image Translation
\cite{conditionalgan} in which a loss is learned by a discriminator network
which is condition on the input. This is further expanded to unpaired image data
using Cyclic GANs \cite{cyclicgan} where the translator attempts to replicate
the distribution seen in the training set. Two translation networks are used,
one in either direction, and cyclic consistency is enforced using the loss
function. The authors claim that this allows the model to learn a sensible mapping between the two domains without the need for a large library of paired data.
