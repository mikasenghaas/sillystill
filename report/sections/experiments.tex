\subsection{Experiments}
\label{subsec:experiments}

The training procedure is illustrated in Figure~\ref{fig:training-overview}. We train our model on a dataset of paired digital-film images. As our image dataset is relatively small, we use a patched training approach, where we randomly sample patches of size $256 \times 256$ from the images before feeding them into the model. We train our model using the Adam optimizer with a learning rate of $1e-3$. We use a batch size of 1 and train the model on 400 patches per image.

The model is trained using the Adam optimizer with a learning rate of $1e-3$. Our experiments can be divided into two parts: single-image experiments and full dataset experiments.

% Single image experiments
For our single-image experiments, we handpicked a simple image in a laboratory
setting showing a white-wall and bright white light source. The image was chosen
as it is perceptually uniform, but contains a variety of visual effects, such as
graininess and colour cast. For these experiments, we train and evaluate the
model on the same image to ensure to observe whether it can overfit and memorize
the desired visual effects. The purpose of this first step is to get an
understanding of the interplay between the loss functions and the model's
ability to learn the desired visual effects in a more controlled setting.

% Full dataset experiments
For the most promising candidates from the single-image experiments, we then
move on to training on the full dataset. The full dataset is split into a
training, validation and test set using a 70-20-10 split ratio. 

% Adding noise channel
For each experiment, we also investigate whether seeding the model with an extra
noise channel improves the model's ability to learn the desired visual effects. This is done by adding a random uniform noise channel to the input images.

Furthermore, we observed that patched training is very stochastic as each patch only captures a very small portion of the original image and can be very different from the rest of the image. To contain more information in each patch we also experiment applying a resized crop where we first crop the image to a larger patch and then resize to the desired input size. This way we can ensure that each patch contains more information from the original image.
